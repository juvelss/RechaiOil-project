User Registrasi & Login

  • Registrasi mudah & cepat
  • Verifikasi KTP/NIB
  • Dashboard personal
2

Setor Minyak Jelantah

User mencatat jumlah minyak melalui aplikasi dan Mitra pengumpul melakukan verifikasi

  • Catat volume minyak
  • Request penjemputan
  • Verifikasi kualitas
3

Pencatatan Blockchain

Data transaksi (volume, user, lokasi) disimpan dalam blockchain agar transparan

  • Data tidak bisa dimanipulasi
  • Transparansi penuh
  • Audit trail lengkap
4

Reward Token

Setelah verifikasi, user otomatis menerima token sesuai jumlah minyak yang disetorkan

  • Token otomatis masuk
  • Real-time notification
  • Bisa ditukar kapan saja
5

Dashboard Admin

Admin memantau pengumpulan minyak, distribusi, dan total token yang dikeluarkan

  • Monitoring real-time
  • Laporan analytics
  • Management system
6

Distribusi ke Industri

Minyak yang terkumpul dijual ke industri biodiesel/kimia sebagai sumber pendapatan

  • Kerjasama industri
  • Harga kompetitif
  • Revenue sharing
